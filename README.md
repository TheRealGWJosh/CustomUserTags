# Installation
### Quick Import
``` 
@import url('https://therealgwjosh.github.io/CustomUserBadges/main/main.css');
```
> Add This To The ***Top of Your Theme*** For a Quick Import And Save Then Enjoy :D
### Powercord & Vizality
